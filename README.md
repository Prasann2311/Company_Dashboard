# 🏢 Company Management Dashboard

A full-stack company management dashboard built with React and MySQL.

## ✨ Features
- 📊 Real-time statistics dashboard
- ➕ Add, Edit, and Delete companies
- 📈 View company details
- 🎨 Modern, responsive UI

## 🛠️ Tech Stack
- **Frontend:** React.js, Axios, Chart.js
- **Backend:** Node.js, Express.js, MySQL
- **Database:** MySQL

## 🚀 Quick Start
```bash
# Backend
cd backend
npm install
npm run dev

# Frontend
cd frontend
npm install
npm start
